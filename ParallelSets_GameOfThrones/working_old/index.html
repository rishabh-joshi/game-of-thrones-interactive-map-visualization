Please explore the data! Category labels can be moved vertically and sub-labels moved horizontally to better see relationships. Using the drop-down menu buttons on top allows you to select what data types are included to help you focus on specific relationships and explore the data.

The band width as well as numbers and % in the tooltips (hover over a band to see them) are based on the numbers of battles that meet each category.

Inspiration

This visualization is based heavily on Jason Davies d3-parasets project. You can find a longer discussion on add-ons I want to build into this d3.js parallel sets project at this blog post. One improvement I added was the ability of the user to select which data categories or dimensions are shown. Additionally, the user can limit the visualizaion to only battles where a specific value in a specific category is present. For example, only the battles where the defender was the House Stark.
