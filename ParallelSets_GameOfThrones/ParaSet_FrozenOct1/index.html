Parallel Sets Data Visualization

by Justin Gosses

Interactive visualisation technique for multidimensional categorical data.

Game of Thrones Battles (298-300)

Please explore the data! Category labels can be moved vertically and sub-labels moved horizontally to better see relationships. Using the drop-down menu buttons on top allows you to select what data types are included to help you focus on specific relationships and explore the data.

The band width as well as numbers and % in the tooltips (hover over a band to see them) are based on the numbers of battles that meet each category.


This a draft version. If you find any bugs or areas for improvement, please submit a pull request on the github repo.

This visualization is based heavily on Jason Davies d3-parasets project. The first advance is the ability of the user to select which data dimensions are shown. Second, using the drop-down menu that extends to the right from the first drop-down menu, the user can limit the visualizaion to battles where a specified dimension matches a specified value. For example, only the battles where the defender was the House Stark or House Lannister.

The dataset for this visualization comes from Kaggle, a data science website geared to competition and community. I suspect there might be at least one error in the original dataset, so check the original data source to see if there are any updates before you use it. You can also download the lightly edited version I used here.
